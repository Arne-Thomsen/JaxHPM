# JaxHPM
[![arXiv](https://img.shields.io/badge/arXiv-2510.26593-b31b1b.svg)](https://arxiv.org/abs/2510.26593)

Code repository for [**Hybrid Physical-Neural Simulator for Fast Cosmological Hydrodynamics**](https://arxiv.org/abs/2510.26593), accepted to the NeurIPS 2025 Workshop on Machine Learning and the Physical Sciences (ML4PS).

![](data/figures/NeurIPS_JaxHPM_Poster.png)

## Installation

### Installing JAX

JaxHPM requires JAX. Install the appropriate version for your system, see the [JAX installation guide](https://jax.readthedocs.io/en/latest/installation.html).

### Basic Installation

```bash
git clone https://github.com/Arne-Thomsen/JaxHPM.git
cd JaxHPM
pip install -e .
```
